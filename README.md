# Rita och gissa

En fristående statisk webbsida med tre knappar:

- Harriet
- Lo
- Vuxna

Varje knapp har 500 möjliga rituppdrag. Sidan fungerar utan byggsteg eller externa beroenden.

## Publicera med GitHub Pages

1. Lägg `index.html` i roten av repot.
2. Commit och push till `main`.
3. Aktivera GitHub Pages från `main` och `/(root)`.
